# Telegram Bot Automator v2

Это вторая версия автоматизатора для Telegram-бота, который использует распознавание изображений для взаимодействия с анкетами.

## Основные возможности

- **Автоматическое нажатие на кнопку "сердце"**: Скрипт находит на экране изображение кнопки (`heart_button.png`) и кликает по нему.
- **Обнаружение и пропуск рекламы**: Использует OCR (Tesseract) для поиска ключевых слов ("Пропустить", "Ладно") и автоматически закрывает рекламные окна.
- **Адаптивность**: Не зависит от точного расположения элементов на экране, что делает его более надежным.
- **Безопасная остановка**: Программу можно в любой момент остановить, нажав клавишу 'q'.

## Требования

- Python 3.x
- Tesseract OCR (должен быть добавлен в PATH)
- Зависимости, перечисленные в `requirements.txt` (включая `opencv-python`).

## Установка

1.  **Установите Tesseract OCR**: Скачайте и установите с [официального сайта](https://github.com/tesseract-ocr/tesseract). Убедитесь, что добавили его в системную переменную PATH.
2.  **Установите зависимости Python**: Запустите файл `install_v2.bat` или выполните команду:
    ```
    pip install -r requirements.txt
    ```

## Использование

1.  **Подготовьте изображение кнопки**: Сделайте скриншот кнопки "сердце" в боте и сохраните его как `heart_button.png` в той же папке, где находится скрипт.
2.  **Запустите Telegram**: Откройте десктопный клиент Telegram и перейдите в чат с ботом.
3.  **Запустите скрипт**: Запустите `run_v2.bat` или выполните в терминале:
    ```
    python automator_v2.py
    ```
4.  **Остановка**: Для завершения работы программы нажмите клавишу 'q'.

## Конфигурация

Все основные настройки находятся в файле `config.py`. Вы можете изменить временные интервалы, чувствительность распознавания изображений и другие параметры.
